### All Features
- Style The Canvas
```js
canvas(engine, width, height, background);
```

- Create A Rectangle With:
```js
createRectangle(engine, backgroundColor, x, y, width, height);
```

- Create A Clickable Object With:
```js
createClickable(backgroundColor, x, y, width, height, func);
```
